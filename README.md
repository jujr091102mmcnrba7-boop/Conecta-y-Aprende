# Conecta y Aprende

Proyecto web educativo construido con Flask.

## Estructura del repositorio

- `proyecto_p1/`: código fuente de la aplicación Flask.
- `MANUALES/`: documentación y manuales del proyecto.
- `templates/`: plantillas HTML.
- `static/`: archivos CSS, imágenes e íconos.

## Cómo ejecutar el proyecto

1. Crear un entorno virtual:
   ```bash
   python -m venv .venv
   ```
2. Activar el entorno virtual:
   - Windows: `.
   .venv\Scripts\Activate`
3. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```
4. Ejecutar:
   ```bash
   python proyecto_p1/app.py
   ```

## Notas

- Ignora `.venv/`, archivos temporales de Python y la carpeta `proyecto_p1/static/uploads/`.
- Este repositorio está preparado para control de versiones y trabajo colaborativo.
